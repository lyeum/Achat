# 커스터마이징 — 이미지 규격 및 등록 가이드

> 설정 패널 → **표정 / 아이콘 지정** / **캐릭터 커스텀** 두 메뉴로 구성된다.

---

## 1. 표정 / 아이콘 지정

### 1-1. 전신 아이콘

캐릭터 대표 이미지. 풀창과 PIP 모드 모두에 사용된다.

| 항목 | 내용 |
|---|---|
| **저장 경로** | `ui_ux/assets/icons/{캐릭터ID}/{캐릭터ID}.png` |
| **권장 해상도** | 512 × 640 px (비율 4:5) |
| **형식** | PNG 권장 (JPG 가능, 투명 배경 불필요) |
| **표시 크기** | 풀창 128×160 px · PIP 160×160 px |
| **렌더 방식** | PreserveAspectFit + **1.2배 확대** + 상단 기준 하단 크롭 |

> 1.2배 확대 후 하단이 잘리므로 **얼굴은 원본 상단 70% 안**에 배치할 것.

---

### 1-2. 감정 레이어

전신 아이콘 위에 겹쳐지는 오버레이 이미지.
표정 변화 부분(눈·입·눈썹 등)만 그린 레이어여야 한다.

| 항목 | 내용 |
|---|---|
| **저장 경로** | `ui_ux/assets/icons/{캐릭터ID}/emotion/{감정}.png` |
| **기본 이미지 경로** | `ui_ux/assets/icons/default/emotion/{감정}.png` |
| **권장 해상도** | 전신 아이콘과 **동일한 캔버스 512 × 640 px** |
| **형식** | **PNG (투명 배경 필수)** |
| **렌더 방식** | 전신 아이콘과 동일 크기로 겹쳐서 합성 |

> 전신 아이콘이 없는 경우 `default/emotion/` 이미지를 대신 표시한다.
> 개별 캐릭터 이미지가 없으면 default → 이모지 플레이스홀더 순으로 fallback.

#### 감정 종류 (9종)

| 파일명 | 감정 | 플레이스홀더 이모지 |
|---|---|---|
| `neutral.png`      | 기본   | 😐 |
| `happy.png`        | 행복   | 😊 |
| `affectionate.png` | 애정   | 🥰 |
| `touched.png`      | 감동   | 🥹 |
| `curious.png`      | 호기심 | 🤔 |
| `sad.png`          | 슬픔   | 😢 |
| `embarrassed.png`  | 당황   | 😳 |
| `annoyed.png`      | 짜증   | 😤 |
| `angry.png`        | 분노   | 😠 |

## 2. 캐릭터 커스텀 (파츠)

픽크루 방식으로 카테고리별 파츠를 조합해 캐릭터를 구성한다.
모든 파츠는 **동일한 캔버스 크기**여야 레이어가 정확히 정렬된다.

| 항목 | 내용 |
|---|---|
| **권장 해상도** | 전 카테고리 공통 **512 × 640 px** |
| **형식** | **PNG (투명 배경 필수)** — base 포함 전체 |
| **렌더 방식** | PreserveAspectFit + 1.2배 확대 + 상단 기준 하단 크롭 |

### 2-1. 레이어 합성 순서

아래에서 위 순서로 겹쳐 합성된다.

| 순서 | 카테고리 | 저장 경로 | 내용 |
|---|---|---|---|
| 1 | `base`    | `ui_ux/assets/characters/base/*.png`    | 얼굴 윤곽·몸통 베이스 (최하단) |
| 2 | `cloth`   | `ui_ux/assets/characters/cloth/*.png`   | 의상 |
| 3 | `hair`    | `ui_ux/assets/characters/hair/*.png`    | 헤어 |
| 4 | `eye`     | `ui_ux/assets/characters/eye/*.png`     | 눈동자·쌍꺼풀 등 |
| 5 | `eyebrow` | `ui_ux/assets/characters/eyebrow/*.png` | 눈썹 |
| 6 | `mouth`   | `ui_ux/assets/characters/mouth/*.png`   | 입 (최상단) |

### 2-2. 제작 가이드

- **base** : 윤곽선·피부·몸통 전체를 그린다. 다른 파츠의 기준 레이어.
- **base 외 전체** : base 위에 얹히는 부분만 그리고 나머지는 투명 처리.
- **파일명** : 자유롭되 카테고리 안에서 의미를 알 수 있게 권장.
  예) `hair_short.png`, `hair_ponytail.png`, `eye_round.png`
- **카테고리에 파일이 없으면** UI에서 "없음" 옵션만 표시되고 해당 레이어는 합성에서 제외.

## 3. 전체 요약표

| 구분 | 저장 경로 | 해상도 | 투명 배경 | 형식 |
|---|---|---|---|---|
| 전신 아이콘 | `icons/{id}/{id}.png` | 512×640 | 불필요 | PNG / JPG |
| 감정 레이어 (9종) | `icons/{id}/emotion/{mood}.png` | 512×640 | **필수** | PNG |
| 감정 기본값 (9종) | `icons/default/emotion/{mood}.png` | 512×640 | **필수** | PNG |
| base    | `characters/base/*.png`    | 512×640 | **필수** | PNG |
| cloth   | `characters/cloth/*.png`   | 512×640 | **필수** | PNG |
| hair    | `characters/hair/*.png`    | 512×640 | **필수** | PNG |
| eye     | `characters/eye/*.png`     | 512×640 | **필수** | PNG |
| eyebrow | `characters/eyebrow/*.png` | 512×640 | **필수** | PNG |
| mouth   | `characters/mouth/*.png`   | 512×640 | **필수** | PNG |

> **핵심**: 전신 아이콘(512×640)을 기준 레퍼런스로 삼고,
> 파츠·감정 레이어도 같은 캔버스에 제작하면 위치 정렬이 자동으로 맞는다.
