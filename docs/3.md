## 1. 전체 RAG 파이프라인 개요

```mermaid
flowchart TB
    subgraph SOURCE["소스 문서"]
        MD["rag/sources/world/Seaside.md\n# WorldName\n## section\n### 항목"]
    end

    subgraph INDEX["인덱싱 파이프라인 (앱 시작 시 1회)"]
        PARSE["_parse_world_md()\n섹션 단위 파싱"]
        EMBED_I["bge-m3 임베딩"]
        STORE["ChromaDB upsert\nworld_knowledge 컬렉션"]
    end

    subgraph RUNTIME["런타임 — 매 턴"]
        QUERY["WorldRetriever.query(user_input)\n코사인 유사도 검색"]
        FILTER["threshold 필터\ndistance ≤ 1.0 - threshold"]
        LAYER_B["Layer B 주입\nPromptBuilder._layer_b()"]
    end

    subgraph WORLDNAV["장소 이동 감지 (선택적)"]
        TRIGGER["키워드 필터\n가자 / 가볼까 / 이동하자 ..."]
        EXTRACT["LLM 장소명 추출\nmax_tokens=15"]
        MATCH["YAML acts 매칭"]
        FIND["find_or_create_location()"]
        GEN["LLM 장소 묘사 생성\nmax_tokens=200"]
        SAVE["add_document()\nChromaDB upsert\nsource=generated"]
    end

    MD --> PARSE --> EMBED_I --> STORE
    STORE --> QUERY --> FILTER --> LAYER_B

    TRIGGER -->|포함| EXTRACT --> MATCH
    MATCH -->|YAML 없음| FIND
    FIND -->|RAG 히트| LAYER_B
    FIND -->|RAG 미스| GEN --> SAVE --> LAYER_B
```

## 2. 인덱싱 파이프라인 (`index_world`)

```mermaid
flowchart TD
    A([앱 시작\nconversation/main.py]) --> B["index_world(world_dir, chroma_path, force=False)"]

    B --> C{world_knowledge\n컬렉션 존재?}
    C -- "있음 & force=False" --> D([스킵 — 기존 인덱스 유지])
    C -- "있음 & force=True" --> E["기존 컬렉션 삭제\nclient.delete_collection()"]
    C -- 없음 --> F["컬렉션 생성\nhnsw:space=cosine"]
    E --> F

    F --> G["world/*.md 파일 순회"]

    G --> H{파일 구조 감지\n# 헤더 존재?}

    H -- "신규 구조\n# WorldName\n## section\n### item" --> I["_parse_world_md()\n섹션 단위 파싱"]
    H -- "레거시 구조\n## 최상위 없음" --> J["_chunk_text()\n고정 크기 청킹\nsize=400 / overlap=50"]

    I --> K["items: list[dict]\nworld_id / section / item_title\ncontent / trigger_keywords"]
    J --> L["chunks: list[str]\n400자 고정 크기 청크"]

    K --> M["chunk_id 생성\n{world_id}_{section}_{item_title}\n특수문자 → _"]
    L --> N["chunk_id 생성\n{stem}_{index:03d}"]

    M --> O["docs = 제목 + 내용\nbge-m3 임베딩 후 ChromaDB add"]
    N --> O

    O --> P(["인덱싱 완료\nworld_knowledge 컬렉션"])
```

## 3. Seaside.md 구조 파서 (`_parse_world_md`)

```mermaid
flowchart TD
    subgraph INPUT["입력 — Seaside.md 형식"]
        F1["# seaside_world           ← world_id"]
        F2["## culture                ← section 변경"]
        F3["### 생활 방식             ← item_title 변경"]
        F4["트리거 키워드: [kw1, kw2] ← 메타만 저장, content 제외"]
        F5["내용 텍스트...            ← content 누적"]
        F6["### 다음 항목             ← flush 후 새 항목 시작"]
    end

    subgraph PARSER["파서 동작"]
        P1["line.startswith('# ')\n→ world_id 설정"]
        P2["line.startswith('## ')\n→ _flush() 후 section 변경"]
        P3["line.startswith('### ')\n→ _flush() 후 item_title 변경"]
        P4["트리거 키워드 정규식 매칭\n→ trigger_keywords 저장\n→ content에서 제외"]
        P5["나머지 줄\n→ current_lines 누적"]
    end

    subgraph OUTPUT["출력 — items list[dict]"]
        O1["world_id:         'seaside_world'"]
        O2["section:          'place'"]
        O3["item_title:       '등대'"]
        O4["content:          '방파제 끝에 세워진...'"]
        O5["trigger_keywords: '등대, lighthouse'"]
    end

    INPUT --> PARSER --> OUTPUT
```

## 5. 매 턴 검색 플로우 (`WorldRetriever.query`)

```mermaid
sequenceDiagram
    participant Router as ConversationRouter
    participant Ret as WorldRetriever
    participant EF as bge-m3 임베딩
    participant DB as ChromaDB

    Router->>Ret: query(user_input)

    Ret->>DB: list_collections()
    DB-->>Ret: 컬렉션 목록

    alt world_knowledge 없음
        Ret-->>Router: [] (빈 리스트)
    end

    Ret->>DB: get_collection("world_knowledge")
    Ret->>EF: 쿼리 텍스트 임베딩
    EF-->>Ret: float 벡터

    Ret->>DB: query(query_texts, n_results=2,\ninclude=[documents, distances])
    DB-->>Ret: docs[], distances[]

    Note over Ret: cutoff = 1.0 - threshold (기본 0.52)

    loop 각 결과
        alt distance ≤ cutoff
            Ret->>Ret: filtered에 추가
        else distance > cutoff
            Ret->>Ret: 제외
        end
    end

    Ret-->>Router: filtered: list[str]
    Router->>Router: PromptBuilder._layer_b()에 전달
```

## 6. 장소 이동 감지 및 동적 생성 (`world_nav`)

```mermaid
flowchart TD
    A["user_input\n예: '카페 가자'"] --> B["detect_move_intent(user_input, llm)"]

    B --> C{_MOVE_TRIGGERS 키워드\n포함 여부?}
    C -- 미포함 --> D([None 반환 — 이동 없음])

    C -- 포함 --> E["LLM 장소명 추출\nmax_tokens=15\n'장소명:' 형식"]

    E --> F{응답 검증}
    F -- "없음 / 30자 초과 / 빈 값" --> D
    F -- 유효한 장소명 --> G["'카페' 반환"]

    G --> H["router._handle_location()\nYAML acts 매칭 시도\n(location 필드 부분 일치)"]

    H -- YAML 매칭 성공 --> I["session.act_id 변경\n→ Layer B YAML act 사용"]

    H -- YAML 미매칭 --> J["find_or_create_location\n(location_name, world_desc, retriever, llm)"]

    J --> K["WorldRetriever.query('카페')"]
    K -- RAG 히트 --> L["기존 묘사 반환"]
    K -- RAG 미스 --> M["create_location_desc()\nLLM 생성 max_tokens=200\n세계관 배경 + 장소 산문 묘사"]

    M --> N["add_document(\n  doc_id='loc_카페'\n  text='[카페] {묘사}'\n  metadata={source:'generated'}\n)"]
    N --> O["ChromaDB world_knowledge\nupsert → 영속 저장"]
    O --> L

    L --> P["session.location_context = '카페\n{묘사}'"]
    P --> Q["Layer B에서\nYAML act 대신\nlocation_context 우선 사용"]
    I --> Q
```

## 7. Layer B 내 RAG 결과 주입 방식

```mermaid
flowchart TD
    subgraph LAYERB["PromptBuilder._layer_b() 조립 순서"]
        B1["[세계관]\n{world.description}"]
        B2{session.location_context\n있음?}
        B2Y["[현재 상황]\n{location_context}\n← 동적 장소 묘사 우선"]
        B2N{YAML act\n있음?}
        B2NY["[현재 상황 — {location}]\n{act.context}"]
        B2NN["(상황 블록 없음)"]
        B3{rag_results\n있음?}
        B3Y["[세계관 배경 — 대화와 관련된 배경 정보]\n- {청크1}\n- {청크2}\n재가공 지시문 포함"]
        B3N["(RAG 블록 없음)"]
    end

    B1 --> B2
    B2 -- Yes --> B2Y
    B2 -- No --> B2N
    B2N -- Yes --> B2NY
    B2N -- No --> B2NN
    B2Y & B2NY & B2NN --> B3
    B3 -- Yes --> B3Y
    B3 -- No --> B3N

    B3Y & B3N --> OUT["Layer B 최종 텍스트\n( \\n\\n join )\n~200 tok → system 메시지에 삽입"]
```

## 8. 재인덱싱 플로우 (소스 수정 후 반영)

```mermaid
flowchart TD
    A["Seaside.md 내용 수정"] --> B["force=True 재인덱싱 실행"]

    B --> C{동적 생성 장소\n보존 필요?}

    D --> E["필요 항목을 Seaside.md의\n## place / ### 항목으로 수동 이전"]

    subgraph WARN["주의"]
        W1["source='generated' 항목은\nSeaside.md에 없음\n→ force=True 시 함께 삭제됨"]
```

## 9. RAG 관련 설정값 (`config.py`)

| 설정 키 | 기본값 | 설명 |
|---|---|---|
| `chroma_path` | `./chroma_deploy` | ChromaDB 저장 경로 |
| `embedding_model` | `BAAI/bge-m3` | 임베딩 모델 |
| `embedding_device` | `cpu` | 임베딩 연산 디바이스 |
| `rag_threshold` | → `vdb_threshold` 폴백 | RAG 검색 threshold (없으면 vdb_threshold 사용) |
| `vdb_threshold` | `0.52` | cosine 유사도 threshold (distance ≤ 1.0 - 0.52 = 0.48) |
| `vdb_top_k` | `2` | 검색 결과 최대 반환 수 |
